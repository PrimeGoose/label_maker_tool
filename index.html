How to Utilize the Rack Label Tool Effectively:

Preparing & Uploading the Excel File:

  1. Prepare an Excel file; the tool will only read locations from column "A". You are free to use other columns for your organizational purposes, such as noting down completed and pending locations.
  2. Ensure that the check digit precedes the location data in your entries. For instance, a valid input can be "A1E DD 01 A1". You can choose to either include or exclude spaces in the data.
  3. if the data is icomplete, then the label pair for that specific location will not be generated.
  4. Save your file and upload it using the 'Upload File' button.

Editing Labels:

Once uploaded, you have the freedom to edit each label individually on the start screen. This functionality is crucial when you wish to apply specific changes to a particular label.

Personalizing Label Appearance:

The tool offers a palette of yellow shades for your labels, alongside an option to adjust the alpha level of the selected yellow, allowing for controlled transparency and a tailored appearance to suit your preferences.

Automated Label Pairing:

The tool automates the process of pairing labels, hence the sequence of locations in your Excel file is not a concern. The application smartly matches and groups all locations to create respective label pairs, simplifying your workflow considerably.
