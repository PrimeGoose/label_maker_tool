How to Use the Rack Label Tool:

Creating & Uploading Excel File:

  1. Prepare an Excel file with a single column.
  2. Enter the header "Label" in cell A1.
  3. List down all your locations in the subsequent cells (A2, A3, and so on).
  4. Save the file and use the 'Upload File' button to upload it to the tool.

Editing Labels:

On the start screen, you can edit each label individually. This is especially useful if you want to make specific changes to a particular label.

Customizing Label Appearance:

Choose from a range of yellow shades for your labels. Additionally, the tool provides an option to adjust the alpha level of the yellow, letting you determine its transparency for a more personalized appearance.
