RackLabelTool

How to Use Excel File

Place all the label data in column 'A' using the following format: [check digit] [location].

Example: 

    UWU DD 23 A2
    ARA DD 23 A1

Important Points to Note

  • Separators like spaces or commas are not required.
  • Case sensitivity is not important; the app will handle it.
  • The items don't have to be in any particular order.
  • There needs to be A1 and A2 pair to produce a label.
  • The application will automatically match a label pairs

How to Use: Printing

To print with the system dialog, use the following shortcuts:

  • Mac: Command+Option+P
  • Windows: Ctrl+Shift+P

Choosing Format

  • A4/7 Button: Using minimal margins, this setting yields 7 180x40mm labels per A4 page.
  • A4/6 Button: Using minimal margins, this setting yields 6 192x46.5mm labels per A4 page.
  • A4/5 Button: Using minimal margins, this setting yields 5 200x56mm labels per A4 page.
  • A4/3 Button: Using minimal margins, this setting yields 3 252x56mm labels per A4 page in Landscape mode. It is meant to be printed in landscape mode for best results.
